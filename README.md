# HA Sentinel Add-on Repository

Dieses Repository enthält **HA Sentinel**, einen lokalen KI-Hausanalysten und Telegram-Steuerbot für Home Assistant.

## Installation

1. In Home Assistant **Einstellungen → Add-ons → Add-on-Shop** öffnen.
2. Oben rechts **Repositories** auswählen.
3. Diese Repository-Adresse hinzufügen:

```text
https://github.com/lybserv2/ha-sentinel
```

4. **HA Sentinel** installieren und konfigurieren.

## Datenschutz

Zugangsdaten, Telegram-Token und persönliche Serveradressen gehören ausschließlich in die lokalen Add-on-Optionen und werden nicht in diesem Repository gespeichert.

## Aktuelle Version

- HA Sentinel 0.5.0
